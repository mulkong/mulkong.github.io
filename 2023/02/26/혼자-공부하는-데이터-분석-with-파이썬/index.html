    혼자_공부하는_데이터_분석_with_파이썬

    작성일 2023-02-26 Edited on 2023-03-05 In 도서 리뷰

    한빛 미디어 서평단 활동의 일환으로 책을 제공 받고 작성된 리뷰입니다.


    이 책을 데이터 분석을 처음 시작하는 사람들에 추천합니다.

    혼자 공부하는 데이터 분석 with 파이썬은 파이썬을 사용하여 데이터 분석에 대한 기초부터 최신 기술까지 다루는 초보자를 위한 책입니다. 쉬운 예제와 연습문제를 통해 독학이 가능하며, 실제 데이터를 사용하여 학습할 수 있습니다. 또한 데이터 시각화에 대해 자세히 다루며, 비전공자도 쉽게 이해할 수 있도록 설명이 잘 되어 있습니. 하지만, 전문적인 내용은 다루지 않기 때문에 실무에서 쓰이는 고급 분석 스킬같은 내용을 원하는 사람들에게는 조금 아쉬울 수 있는 책일거라 생각이 듭니다. 따라서 이 책은 데이터 분석을 처음 시작하는 사람들에게 추천합니다.


    이 책을 읽는 방법은 다음과 같습니다.

    파이썬 기초 학습

    이 책은 파이썬을 사용하여 데이터 분석을 다루기 때문에, 파이썬에 익숙하지 않은 경우, 파이썬 기초부터 학습하는 것이 좋습니다.

    구성 파악

    이 책은 구성이 체계적이며, 초보자가 따라하기 쉬운 예제와 연습문제가 많이 수록되어 있습니다. 먼저, 책의 전반적인 구성과 각 장의 내용을 파악하면서 공부 계획을 세우는 것이 좋습니다.

    예제 실습

    각 장에서 제공하는 예제와 연습문제를 실습해보면서, 이론을 이해하고 코드 작성 능력을 향상시킬 수 있습니다. 예제를 따라하면서 스스로 코드를 수정해보는 것도 좋은 방법입니다.

    데이터 활용

    이 책에서는 실제 데이터를 사용하여 데이터 분석을 다루고 있습니다. 따라서, 예제에서 사용한 데이터뿐만 아니라 다른 데이터를 활용하여 실습해보는 것도 좋은 방법입니다.

    데이터 시각화 실습

    데이터 시각화는 데이터 분석 결과를 전달하기 위해 매우 중요한 기술입니다. 이 책에서는 데이터 시각화에 대해 상세히 다루고 있으므로, 데이터 시각화를 배우고 실습해보는 것도 좋은 방법입니다.

    추가 학습

    이 책은 데이터 분석을 처음 시작하는 초보자를 대상으로 한 책이기 때문에, 전문적인 내용은 다루지 않습니다. 따라서, 데이터 분석에 대한 추가 학습이 필요한 경우, 이 책에서 소개한 기술들을 더 깊이 있게 학습하는 것이 좋습니다. 관련 온라인 강의나 다른 책을 참고하는 것도 좋은 방법입니다.
